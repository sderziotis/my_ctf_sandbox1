# Lateral Movement Lab

## Scenario

You have gained access to a workstation inside a corporate LAN using weak SSH credentials.

Your objectives:
1. Perform network discovery
2. Identify internal services
3. Extract credentials or keys
4. Move laterally to the internal server
5. Access the internal Flask application

## Network

- Attacker: 192.168.0.3
- Workstation: 192.168.0.10
- Server: 192.168.0.2

## Tools Available

- nmap
- hydra
- arp-scan
- ssh
